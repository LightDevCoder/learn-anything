# learn-anything (legacy source)

> **Migration notice:** The current authoritative installation source is the
> released [LightDevCoder/skills](https://github.com/LightDevCoder/skills/releases/tag/v0.1.0)
> first-party Skills Collection.

This repository is the preserved legacy source for `learn-anything`. The
maintained, independently installable package is now released in the
first-party collection. Keep this checkout for its history and tags; use the
new collection for current installation and documentation.

## Install the current Skill

Install only `learn-anything`:

```text
npx skills add LightDevCoder/skills --skill learn-anything
```

Or install the complete stable first-party collection:

```text
npx skills add LightDevCoder/skills
```

Read the collection's [installation guide](https://github.com/LightDevCoder/skills/blob/main/docs/INSTALLATION.md)
for scope, fresh-install verification, and the manual fallback.

The released package includes the Skill contract, hooks, agents, and referenced
resources. Do not copy only `SKILL.md` when a package resource is needed.

## Source boundaries

This repository is no longer the current first-party installation authority.
Direct-use Matt Pocock Skills remain at their [upstream repository](https://github.com/mattpocock/skills)
and are not copied into the collection. Modified third-party Skills belong in
the separate [private governance repository](https://github.com/LightDevCoder/skills-3rdParty).

## History and archival

This repository is not archived by this redirect step. Old Git history and tags
remain preserved here. Any archival action will occur only after this redirect
has been verified and will leave the migration guidance available in the
archived repository.
