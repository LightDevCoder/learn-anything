# learn-anything（旧版源码仓库）

> **迁移通知：** 当前权威安装来源是已发布的
> [LightDevCoder/skills](https://github.com/LightDevCoder/skills/releases/tag/v0.1.0)
> 第一方 Skills Collection。

本仓库保留 `learn-anything` 的旧版源码、Git 历史和标签。当前可安装、可
维护的 Skill 已迁移到第一方集合。请使用新的集合进行安装和查阅最新文档。

## 安装当前 Skill

只安装 `learn-anything`：

```text
npx skills add LightDevCoder/skills --skill learn-anything
```

或安装完整的稳定第一方集合：

```text
npx skills add LightDevCoder/skills
```

请阅读新的[安装指南](https://github.com/LightDevCoder/skills/blob/main/docs/INSTALLATION.md)，
了解安装范围、全新环境验证和手动回退方式。

未修改的 Matt Pocock Skills 仍由其[上游仓库](https://github.com/mattpocock/skills)
负责，不会为了方便而复制到第一方集合。修改过的第三方 Skill 属于单独的
[私有治理仓库](https://github.com/LightDevCoder/skills-3rdParty)。

本次重定向不会归档本仓库。旧历史和标签会继续保留；只有在重定向完成验证
后才会执行归档，并在归档仓库中保留迁移说明。
